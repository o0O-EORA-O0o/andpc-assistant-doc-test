# Orientation prioritaire fictive OP-001

L'orientation prioritaire OP-001 concerne l'amélioration de la qualité des actions de formation en santé.

Les actions rattachées à cette orientation doivent préciser :

- le public cible ;
- les objectifs pédagogiques ;
- la durée de l'action ;
- les modalités pédagogiques ;
- les critères d'évaluation.

Une action doit être cohérente avec le public ciblé et les objectifs annoncés.

## Public concerné

L'OP-001 peut concerner les professionnels de santé impliqués dans l'amélioration continue de leurs pratiques.

## Points de vigilance

Une action rattachée à l'OP-001 doit permettre d'identifier clairement le lien entre le contenu pédagogique, le public cible et les objectifs déclarés.
