# Fiche de cadrage fictive OP-001

Pour être conforme, une action rattachée à l'OP-001 doit comporter les éléments suivants :

1. un objectif pédagogique principal clairement formulé ;
2. une durée minimale de 3 heures ;
3. une description des modalités pédagogiques ;
4. une méthode d'évaluation des acquis ;
5. une justification de l'adéquation entre le public cible et le contenu.

Les actions sans objectif pédagogique explicite doivent faire l'objet d'un point de vigilance.

## Modalités pédagogiques

Les modalités peuvent être présentielles, non présentielles ou mixtes, à condition qu'elles soient décrites et cohérentes avec les objectifs pédagogiques.

## Critères d'analyse

L'agent doit vérifier la cohérence entre :

- l'orientation prioritaire déclarée ;
- la typologie de l'action ;
- la durée ;
- le programme ;
- les publics professionnels ciblés.
